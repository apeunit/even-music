## Getting Started

### Prerequisite

- [Nodejs](https://nodejs.org/en/download/)

### Installation

Run the development server:

1. Clone the project

```bash
git clone https://github.com/apeunit/even-music.git
```

2. Install packages

```bash
npm install
```

3. Run local server

```bash
npm run dev
```
